Funciones

Es una de las partes más importante de este lenguaje, ya que nos permite reutilizar el código de una forma muy sencilla y eficiente. Esdecir, gracias a las funciones podemos crear una plantilla de código y luego simplemete podemos invocarla las veces que necesitemos para que ejecute ese trozo de código.

Ejemplo :

∫

function suma(num1, num2){

return num1 + num2;

}

function log(data){

console.log(data)

}

log('Hola, Mundo!');

log( suma(1, 2) );

Sintaxis:

Primero, utilizamos la palabra reservada function.

A continuación, asignamos un nombre a esa función.

Entre paréntesis, especificamos los parámetros que la función puede aceptar.

Luego, utilizamos la palabra reservada return seguida de las instrucciones que deseamos ejecutar dentro de la función.

ACLARACIÓN:

Definir una función no la ejecuta. Definir una función simplemente la nombra y especifica qué hacer cuando la función es llamada. Llamar la función es lo que realmente realiza las acciones especificadas con los parámetros indicados. Por ejemplo, si define la función square, podría llamarla como sigue:

square(5);

La sentencia anterior llama a la función con el argumento 5. La función ejecuta sus sentencias y retorna el valor 25.
